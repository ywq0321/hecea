# HECEA Website

## Overview
A modern, responsive website for the UK Higher Education Chinese Employees Association featuring member authentication and differentiated content access.

## Features
- **Public Content**: Information about the organization, services, and community
- **Member Authentication**: Registration and login system
- **Membership Application System**: Comprehensive application form with admin review process
- **Admin Panel**: Review and approve/reject membership applications
- **Member-Exclusive Content**: Integrated member portal within about page
- **Forum System**: Discussion forums for each service area
- **Responsive Design**: Works on desktop, tablet, and mobile devices

## Pages
- `index.html` - Main homepage with organization information
- `about.html` - Comprehensive About page with organization structure, policies, and member portal
- `workplace-rights.html` - Detailed workplace rights protection service page
- `career-development.html` - Career and holistic development service page
- `research-collaboration.html` - Research collaboration service page
- `workplace-rights-forum.html` - Forum for workplace rights discussions
- `career-development-forum.html` - Forum for career & holistic development discussions
- `research-collaboration-forum.html` - Forum for research collaboration discussions
- `admin.html` - Admin panel for reviewing member applications

## Key Components
- **Header**: Navigation with authentication status
- **Hero**: Welcome section with personalized content for members
- **About**: Organization mission and statistics
- **Services**: Workplace rights protection, career & holistic development, research collaboration
- **Community**: Member testimonials and success stories
- **Footer**: Contact information and links
- **AuthModal**: Registration, login, and membership application functionality
- **ForumPost**: Reusable component for creating forum posts
- **AboutSections**: Unified sections including member portal access
- **AdminApp**: Admin panel for reviewing and managing member applications

## Member Benefits
- Professional networking opportunities
- Career development workshops and mentorship
- Research collaboration facilitation
- Workplace rights protection and legal guidance
- Career and holistic development programs
- Cultural exchange events
- Exclusive member resources and updates

## Technology Stack
- React 18 for interactive components
- TailwindCSS for styling
- Local storage for user session management
- Modern responsive design principles
